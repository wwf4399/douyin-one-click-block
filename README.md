# 抖音一键拉黑
一个可以一键拉黑抖音博主的谷歌浏览器插件  
---
**使用方法**：谷歌浏览器->设置->扩展程序->我的扩展程序->开发者模式->加载未打包的扩展程序->[在弹出的文件资源管理器中选择从这里下载的插件源代码]
